\# 💰 Gestão Pro - Sistema de Gestão Financeira PWA



O \*\*Gestão Pro\*\* é uma aplicação web progressiva (PWA) desenvolvida para facilitar o controle financeiro. Focado em agilidade e organização, o sistema permite o acompanhamento detalhado de entradas, saídas e saldos em tempo real, com uma interface limpa e intuitiva.



---



\## 🚀 Funcionalidades Principais



\* \*\*Controle de Fluxo de Caixa:\*\* Registro rápido de receitas e despesas.

\* \*\*Dashboard Dinâmico:\*\* Visualização clara do saldo atual e resumo financeiro.

\* \*\*Interface Responsiva:\*\* Totalmente otimizado para uso em smartphones, tablets e desktops.

\* \*\*Experiência PWA:\*\* Pode ser instalado no dispositivo como um aplicativo, permitindo acesso rápido.

\* \*\*Gestão de Categorias:\*\* Organização inteligente para análise de gastos e ganhos.



---



\## 🛠️ Tecnologias Utilizadas



Este projeto utiliza tecnologias modernas para garantir performance e segurança:



\* \*\*HTML5 \& CSS3:\*\* Estruturação semântica e design responsivo.

\* \*\*JavaScript (ES6+):\*\* Lógica funcional e manipulação dinâmica de dados.

\* \*\*Firebase (Integração):\*\* Utilizado para persistência de dados e autenticação segura.

\* \*\*Service Workers:\*\* Configuração para suporte a PWA e carregamento otimizado.
